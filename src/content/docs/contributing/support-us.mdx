## Why support us?

Accessible Astro Components is an open-source project dedicated to making the web more accessible. Your support helps us:
- Maintain and improve components
- Create new accessible components
- Write better documentation
- Provide community support
- Keep everything free and open source

## Ways to support

- Development time

### Non-financial support

There are many other valuable ways to support the project:

- Star our repositories on GitHub
- Share the project with others
- Report bugs and issues
- Suggest new features
- Contribute code or documentation
- Help others in discussions
